SMART CONTRACT ESSENTIALS
===========================


**Duration : 18JAN - 29JAN**

**Instructor : Dhuvin Parikh**

Course Description
------------------

This course will provide a deeper dive into the development of smart contracts. The focus of this course will be on analyzing applicable use cases and designing token-based solutions. Students will write smart contracts, with an emphasis on Solidity, and develop test suites. They will gain hands-on experience with using current industry accepted development tools and frameworks.
